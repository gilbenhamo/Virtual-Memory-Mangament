Description:

This project is about creating a Memory management simulation that will use the principles
of page fault algorithms
___________________________________________________________________

Environment:

OS: Ubonto os (Linux)

requierments to install:
                gcc compiler

How to compile and run:
    compile:        gcc -o MemoryManagement main.c
    run:            ./MemoryManagement useLru InputFileName OutputFileName m n
    compile+run:    gcc -o MemoryManagement main.c;./MemoryManagement useLru InputFileName OutputFileName m n

    userLru == 1 => the program will use LRU algorithm
    m => size of secondary Memory
    n => size of main Memory

    if file name got a '(' ')' in the name you will have to use "OutputFileName" in the command

___________________________________________________________________

How to Run Your Program:
    
    Example:
        ./MemoryManagement 1 intput.txt "output(LRU).txt" 1000 50

Our program output will be:

    ~ Runnig Memory simulation with LRU Page fault algorithm ~
    free allocated memory
    ~ End memory simulation - Export data to file: output(LRU).txt ~
